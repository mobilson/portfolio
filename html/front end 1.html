Ik ga hiet vertellen over de talen waarover wij gaan leren in deze les.

HTML

HTML staat voor HyperText Markup Language. Dit is een taal die eiglijk het skalet is je web pagina. In HTML schrijf je de tekst die je op je webpagina wil hebben. Ook kan media plaatsten zoals foto's, video's en GIF's.

CSS

CSS staat voor Cascading Style Sheets. Je gebruikt deze taal om je HTML te stylen en het een beetje levenb te geven.

JavaScript

Op dit moment gebruiken wij JavaScript om robots te programmeren, doormiddel van de adafruit.
